
Feature
=======

.. granite:servercomponent:: /libs/granite/ui/components/coral/foundation/renderconditions/feature
   :rendercondition:

   A condition that decides based on :javadoc:`Sling's Feature Flag <org/apache/sling/featureflags/package-summary.html>`.

   It has the following content structure:

   .. gnd:gnd::

      [granite:RenderConditionsFeature]

      /**
       * The feature name.
       */
      - feature (String)
