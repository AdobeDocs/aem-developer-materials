6.7.32 LowerCase

AQM

type LowerCase extends DynamicOperand ::=

DynamicOperand operand


Evaluates to the lower-case string value (or values, if multi-valued) of operand.

If operand does not evaluate to a string value, its value is first converted to a string as described in ยง3.6.4 Property Type Conversion. The lower-case string value is computed as though the toLowerCase() method of java.lang.String were called.

If operand evaluates to null, the LowerCase operand also evaluates to null.
