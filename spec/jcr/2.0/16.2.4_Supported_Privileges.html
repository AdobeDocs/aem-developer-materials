16.2.4 Supported Privileges

The privileges available for a particular node can be determined through

Privilege[]
AccessControlManager.
getSupportedPrivileges(String absPath)

where absPath is the location of the node. Note that this method does not return the privileges held by a Session with respect to the specified node, but rather the privileges supported by the repository with respect to that node (see ยง16.3.7 Testing Privileges).
