8.5.2 Default Primary Node Type

The methods

NodeType NodeDefinition.getDefaultPrimaryType() and

String NodeDefinition.getDefaultPrimaryTypeName()

return information about the default primary node type of the definition (ยง3.7.4.2 Default Primary Node Type). The latter method returns the name of the node type while the former method returns the live NodeType object representing the type. The former only functions if the NodeDefinition is part of a live registered NodeType.
