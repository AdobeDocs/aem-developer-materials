3.2.7 Equality of Names

Two JCR names (N1, L1) and (N2, L2) are equal if and only if N1 is equal to N2 and L1 is equal to L2, according to the definition of string equality used in the String.compareTo method. This definition applies both in the general context of using an API method that takes or returns a JCR name and in the specific case of comparing values of type NAME (see ยง3.6.5.8 NAME).
