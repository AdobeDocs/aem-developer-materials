3.7.11.5 nt:resource

[nt:resource] > mix:mimeType, mix:lastModified

primaryitem jcr:data

- jcr:data (BINARY) mandatory

This node type may be used to represent the content of a file. In particular, the jcr:content subnode of an nt:file node will often be an nt:resource. Note that the definition of this node type indicates multiple inheritance (see ยง3.7.6 Node Type Inheritance).
