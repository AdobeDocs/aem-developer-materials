Accessibility

              

                      
Keyboard support

                  The NumberInput component follows the ARIA specification for spinbutton. This means the following keys are valid
                  for interacting with it: up, down, left, right,
                  pageup, pagedown, home, end and the Mouse Wheel. Go ahead and
                  try them on any component of the page!
                      
Labelling the component

                  For improved accessibility the component should always be labelled. This can be achieved by using the
                  labelledby attribute.

instance.max {Number}
    

    
    Reflected
    

    

    






    The maximum value for the NumberInput. If a value above the maximum is set, the NumberInput will be marked as
        invalid but the value will be preserved. Stepping up the NumberInput via Coral.NumberInput#stepUp or
        the increment button respects the maximum value. It reflects the max attribute to the DOM.






    


    

	

	

	

	

	

	

	

  
  
Default Value:

  
  • null

  

  
  
HTML Attribute:

  
  • max

  

	

  

	

	




    




        
            



    
instance.min {Number}
    

    
    Reflected
    

    

    






    The minimum value for the NumberInput. If a value below the minimum is set, the NumberInput will be marked as
        invalid but the value will be preserved. Stepping down the NumberInput via Coral.NumberInput#stepDown
        or the decrement button respects the minimum value. It reflects the min attribute to the DOM.






    


    

	

	

	

	

	

	

	

  
  
Default Value:

  
  • null

  

  
  
HTML Attribute:
