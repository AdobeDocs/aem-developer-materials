Class CQ_Analytics.Variables

Package:CQ_Analytics
Class:Variables
Extends:Object
Clientlib:personalization.core
A helper class providing a set of ClientContext utility methods to handle variables: a variable is a marker used in a store property value to referenced the value of another property. Format: ${/storeName/propertyName}

Added in CQ 5.5.
