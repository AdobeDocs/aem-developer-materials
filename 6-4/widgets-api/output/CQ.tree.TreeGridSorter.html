Class CQ.tree.TreeGridSorter

Package:CQ.tree
Class:TreeGridSorter
Extends:CQ.Ext.tree.TreeSorter
Clientlib:cq.widgets
Provides sorting of nodes in a CQ.tree.TreeGrid. The TreeGridSorter automatically monitors events on the associated TreeGrid that might affect the tree's sort order (beforechildrenrendered, append, insert and textchange). Example usage:
new CQ.tree.TreeGridSorter(myTreeGrid, {
     folderSort: true,
     dir: "desc",
     sortType: function(node) {
         // sort by a custom, typed attribute:
         return parseInt(node.id, 10);
     }
 });

  sortType : Function
A custom "casting" function used to convert node values before sorting. The function will be called with a single par...
A custom "casting" function used to convert node values before sorting. The function will be called with a single parameter (the CQ.Ext.tree.TreeNode being evaluated) and is expected to return the node's sort value cast to the specific data type required for sorting. This could be used, for example, when a node's text (or other attribute) should be sorted as a date or numeric value. See the class description for example usage. Note that if a sortType is specified, any property config will be ignored.
