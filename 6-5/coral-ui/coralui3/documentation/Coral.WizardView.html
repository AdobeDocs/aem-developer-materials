Coral.WizardView

Current Version: 1.2.1
Related Components: Coral.StepList, Coral.PanelStack
Github: @coralui/coralui-component-wizardview

A Coral.WizardView is the wrapping container used to create a tabbed UI element. This is intended to be used with a Coral.StepList and a Coral.PanelStack to create the typical 'Wizard' pattern.

Wizard Pattern Example

Here is an example of how create a UI 'wizard' in CoralUI. This requires a Coral.WizardView container, which holds a Coral.StepList and a Coral.PanelStack. 

                    
                      
                        Step 1
                        Step 2
                        Step 3
                        Step 4
                      
                      
                        
                          This is the content of the first step.
                          
                        
                        
                          This is the content of the second step.
                          
                          
                        
                        
                          This is the content of the third step.
                          
                          
                        
                        
                          This is the content of the fourth step.
