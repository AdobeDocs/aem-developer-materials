Class CQ.Ext.menu.MenuMgr

Package:CQ.Ext.menu
Class:MenuMgr
Extends:Object
Clientlib:cq.widgets
Provides a common registry of all menu items on a page so that they can be easily accessed by id.

This class is a singleton and cannot be created directly.
