foundation.validation.validator: foundation.jcr.name¶

foundation.jcr.name is a validator of foundation-validation.

Checks the value of the field against valid JCR name. If the value is empty, this validator is voting abstain. This is an equivalence to JcrUtil.html#isValidName(String)
