8.5.1 Required Primary Node Types

The methods

NodeType[] NodeDefinition.getRequiredPrimaryTypes() and

String[] NodeDefinition.getRequiredPrimaryTypeNames()

return information about the required primary node types of the definition (ยง3.7.4.1 Required Primary Node Types). The latter method returns the names of the node types while the former method returns the live NodeType objects representing the types. The former only functions if the NodeDefinition is part of a live registered NodeType.
