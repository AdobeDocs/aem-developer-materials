15.2.1.7 Version Extends Node

The Version interface extends Node. Under simple versioning, however, versions are not represented by nodes in content, consequently the inherited methods, other than Item.getName() (see ยง15.2.1.1 Version Name), are not required to function. These methods may throw a RepositoryException.
