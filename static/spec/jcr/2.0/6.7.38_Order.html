6.7.38 Order

AQM

enum Order ::=

Ascending,

Descending


Order is either Ascending or Descending.

JCR-SQL2

Order ::= Ascending | Descending


Ascending ::= 'ASC'


Descending ::= 'DESC'


JCR-JQOM

An order is a String constant. One of:

QueryObjectModelConstants.JCR_ORDER_ASCENDING

QueryObjectModelConstants.JCR_ORDER_DESCENDING
