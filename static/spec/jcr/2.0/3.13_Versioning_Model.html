3.13 Versioning Model

Versioning enables a user to record the state of a node and its subgraph and restore that state at a later time. A repository that supports versioning may support either the complete set of versioning features, referred to as full versioning, or a specific subset defined here, which is referred to as simple versioning. This section describes the concepts, data structures and node types of the full versioning model as well as which parts of that model apply under simple versioning.
