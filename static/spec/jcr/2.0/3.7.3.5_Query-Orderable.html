3.7.3.5 Query-Orderable

A property may be declared query-orderable, meaning that query results may be ordered by this property using the order clause of a query (see ยง6.7.37 Ordering). The query-orderable flag is a boolean.

3.7.3.5.1 Interaction with Node Type Definition

This attribute only takes effect if the node type holding the property definition has a queryable setting of true (see ยง3.7.1.5 Queryable Node Type), otherwise this attribute is automatically set to false.
