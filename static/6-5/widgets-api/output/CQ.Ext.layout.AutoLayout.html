Class CQ.Ext.layout.AutoLayout

Package:CQ.Ext.layout
Class:AutoLayout
Extends:Object
Clientlib:cq.widgets

The AutoLayout is the default layout manager delegated by CQ.Ext.Container to render any child Components when no layout is configured into a Container.. AutoLayout provides only a passthrough of any layout calls to any child containers.
