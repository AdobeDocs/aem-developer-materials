This class is a singleton and cannot be created directly.

<static> Add an event listener to an obj.listeners config. This is the same as obj.listeners[event] = handler, but will also handle the case if there is already a listener defined for that event. The provided handler will be called when the event is triggered after the function that is already defined in the config object.

This method can be used on the raw config object as read from JSON, it will automatically convert string-based functions using CQ.Util.formatData.

Parameters:
  • obj : Object
    the config object; events are set on the "listeners" child
  • event : String
    name of the event for which the function should listen to
  • handler : Function
    function to listen for the event

  patchText( String text, String/String[] snippets ) : String
<static> Replaces occurrences of {n} in the specified text with the texts from the snippets. Example 1 (single ...
<static> Replaces occurrences of {n} in the specified text with the texts from the snippets.

Example 1 (single snippet):

var text = CQ.Util.patchText("{0} has signed in.", "Jack");
Result 1:
Jack has signed in.

Example 2 (multiple snippets):

var text = "{0} {1} has signed in from {2}.";
text = CQ.Util.patchText(text, ["Jack", "McFarland", "x.x.x.x"]);
Result 2:
Jack McFarland has signed in from x.x.x.x.

Parameters:
  • text : String
    The text
  • snippets : String/String[]
    The text(s) replacing {n}
Returns:
  • String
    The patched text
