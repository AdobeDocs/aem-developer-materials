Class CQ_Analytics.PersistedSessionStore

Package:CQ_Analytics
Class:PersistedSessionStore
Subclasses:ClientContextMgr, PersistedJSONStore, ProfileDataMgr, TagCloudMgr
Extends:CQ_Analytics.SessionStore
Clientlib:personalization.core
A PersistedSessionStore is a persisted container of properties/values. The persistence is done with CQ_Analytics.SessionPersistence.
