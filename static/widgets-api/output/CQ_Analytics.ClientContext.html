Class CQ_Analytics.ClientContext

Package:CQ_Analytics
Class:ClientContext
Extends:Object
Clientlib:personalization.clientcontext
The ClientContext object is a facade to access / update the session stores and their properties contained in the CQ_Analytics.ClientContextMgr.


This class is a singleton and cannot be created directly.
Added in CQ 5.5.
