Deprecated: use GraniteTiming instead

A helper class providing load time statistics.

To enable go to Apache Felix Web Management Console > Configuration > CQ HTML Library Manager and enable "Timing". To show the statistics hit [ctr][shift][u]. Note that the logs are not added continuously but before they are displayed. Therefore wait until the page has been loaded before displaying the logs.

In Content Finder set the focus to the content page before displaying the logs in order to display the statistics of the content page.



This class is a singleton and cannot be created directly.
