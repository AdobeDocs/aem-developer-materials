Class CQ_Analytics.Operator

Package:CQ_Analytics
Class:Operator
Extends:Object
Clientlib:personalization.core
The CQ_Analytics.Operator object is a singleton providing the most common operator names and utils around the operators.

This class is a singleton and cannot be created directly.

  getOperation( CQ_Analytics.Operator operator ) : String/Function
Returns operator operation, which can be either:
  • String: mathematical JS operator like ==, <, <=, > ...
  • Function: function requiring 2 parameters, the 2 values to operate and which returns true if operation success, false otherwise. Example: contains operator function. function(s1, s2) { if (s1) { if (s2) { s1 = "" + s1; s2 = "" + s2; return s1.toLowerCase().indexOf(s2.toLowerCase()) != -1; } return true; } return false; }
