Gets the globally shared flyweight Element, with the passed node as the active element. Do not store a reference to this element - the dom node can be overwritten by other code. Shorthand of CQ.Ext.Element.fly

Use this to make one-time references to DOM elements which are not going to be accessed again either by application code, or by Ext's classes. If accessing an element which will be processed regularly, then CQ.Ext.get will be more appropriate to take advantage of the caching provided by the CQ.Ext.Element class.

  • named : String
