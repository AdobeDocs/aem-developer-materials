Class CQ.Ext.WindowMgr

Package:CQ.Ext
Class:WindowMgr
Extends:CQ.Ext.WindowGroup
Clientlib:cq.widgets
The default global window group that is available automatically. To have more than one group of windows with separate z-order stacks, create additional instances of CQ.Ext.WindowGroup as needed.

This class is a singleton and cannot be created directly.

  register( Window win ) : void
Registers a Window with this WindowManager. This should not need to be called under normal circumstances. Windows are...

Registers a Window with this WindowManager. This should not need to be called under normal circumstances. Windows are automatically registered with a manager at construction time.

Where this may be useful is moving Windows between two WindowManagers. For example, to bring the CQ.Ext.MessageBox dialog under the same manager as the Desktop's WindowManager in the desktop sample app:

var msgWin = CQ.Ext.MessageBox.getDialog();
MyDesktop.getDesktop().getManager().register(msgWin);

    The Window to register.
