
Heading
=======

.. granite:servercomponent:: /libs/granite/ui/components/coral/foundation/heading

   A heading component to represent HTML h1 - h6.

   It has the following content structure:

   .. gnd:gnd::

      [granite:Heading] > granite:commonAttrs, granite:renderCondition

      /**
       * The heading level.
       */
      - level (Integer) = 1 < 1, 2, 3, 4, 5, 6

      /**
       * The heading text.
       */
      - text (String) i18n
