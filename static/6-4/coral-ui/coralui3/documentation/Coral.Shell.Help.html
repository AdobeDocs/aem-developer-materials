Coral.Shell.Help

Current Version: 2.1.4
Github: @coralui/coralui-component-shell-help

The Coral.Shell.Help component is a standardized help menu designed to fit into the Shell 3 pattern. The menu typical contains links that are instances of Coral.Shell.Help.Item.
