  onStoreInitialized( String storeName, Function callback, Boolean/Number delay ) : void
<static> Executes a callback function when a store is initialized. Some store might be initialized several time...
<static> Executes a callback function when a store is initialized. Some store might be initialized several times (default init + asynchronous data loading): use delay parameter to try to get only one call of the callback. A timeout is defined before every init event, next init event kills all previous calls. Latest one execute the callback. WARNING: if time between 2 init events is bigger than timeout, callback is called several times.
Parameters:
  • storeName : String
    Name of the store
  • callback : Function
    Function to execute
  • delay : Boolean/Number
    True to enable a default delay (value is DEFAULT_INIT_DELAY or a number of milliseconds (defaults false).
