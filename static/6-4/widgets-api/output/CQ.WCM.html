  getDialog( String/Object config, [String key], [Boolean nocreate] ) : CQ.Dialog
<static> Creates and/or returns a Dialog using the specified configuration (which may also be a URL serving one...
<static> Creates and/or returns a Dialog using the specified configuration (which may also be a URL serving one of the configurations below).

Single Panel:


"jcr:primaryType": "cq:Panel",
items: {
    "jcr:primaryType": "cq:WidgetCollection",
    field1: {
        fieldLabel: "Title",
        name: "./jcr:title",
    }
}

Add a title to create a TabPanel with a single Panel:


"jcr:primaryType": "cq:Panel",
title: "Tab 1"
items: {
    ...
}

An entire TabPanel:


"jcr:primaryType": "cq:TabPanel",
items: {
    "jcr:primaryType": "cq:WidgetCollection",
    tab1 : {
        "jcr:primaryType": "cq:Panel",
        title: "Tab 1",
        items: {
            "jcr:primaryType": "cq:WidgetCollection",
            field1: {
                ...
            }
        }
    }
    tab2: {
        ...
    }
}

An entire Dialog (e.g. with a Tab Panel):


"jcr:primaryType": "cq:Dialog",
items: {
    "jcr:primaryType": "cq:TabPanel",
    items: {
        "jcr:primaryType": "cq:WidgetCollection",
        tab1 : {
            "jcr:primaryType": "cq:Panel",
            title: "Tab 1",
            items: {
                ...
            }
        }
    }
}

The entire dialog config accepts all dialog specific config options:


"jcr:primaryType": "cq:Dialog",
okText: "Custom OK",
cancelText: "Custom Cancel",
buttons: CQ.Dialog.CANCEL,
params: {
    hiddenField1: "hiddenValue1",
    hiddenField2: "hiddenValue2"
}

Parameters:
  • config : String/Object
    The URL to retrieve the dialog config from or a config object.
  • key : String
    (optional) The key to cache the dialog under (set to "NEW" to avoid dialog caching)
  • nocreate : Boolean
    (optional) True to only look for dialog in cache
Returns:
  • CQ.Dialog
    The dialog

If a component is used inside a CQ.Dialog, you must not use this method, as the component is automatically detected as a valid drop target.

Note that you should always explicitly unregister any component you previously registered.

Parameters:
  • dropTargetComp : CQ.Ext.Component
    The drop target component to register
Returns:
  • void
