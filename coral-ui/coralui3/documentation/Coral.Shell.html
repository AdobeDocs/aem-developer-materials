Coral.Shell

Current Version: 4.2.2
Github: @coralui/coralui-component-shell

The Coral.Shell component is a wrapping container for the coralui-shell-* components.

See also:

  • Coral.Shell.Header
  • Coral.Shell.MenuBar
  • Coral.Shell.User
  • Coral.Shell.OrgSwitcher
  • Coral.Shell.SolutionsSwitcher
  • Coral.Shell.Workspaces

An example of full Shell 3 in action can be found in the Shell 3 documentation.

Coral.Shell API

new Coral.Shell() or document.createElement('coral-shell')
