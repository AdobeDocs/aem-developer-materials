Coral.AnchorButton

Current Version: 1.5.1
Related Components: Coral.Button
Github: @coralui/coralui-component-anchorbutton

A anchorbutton containing text and/or an icon. It extends HTMLAnchorElement to take advantage of the accessibility given by browsers but it means that this component has a special way of instantiating.

Accessibility

Keyboard support

While it is preferable to use the Coral.Button component to include a button element in an application, Coral.AnchorButton allows us to style an anchor element that both looks and behaves like a button rather than a link. Coral.AnchorButton can receive keyboard focus regardless of whether or not it has an href attribute, can be activated using either the SPACE key or the ENTER key, and is identified to assistive technology as a button element.

Coral.AnchorButton API

JavaScript:

new Coral.AnchorButton() or document.createElement('a', 'coral-anchorbutton')
