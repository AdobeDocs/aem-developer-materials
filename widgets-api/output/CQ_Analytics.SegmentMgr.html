Class CQ_Analytics.SegmentMgr

Package:CQ_Analytics
Class:SegmentMgr
Extends:CQ_Analytics.SessionStore
Clientlib:personalization.core
The CQ_Analytics.SegmentMgr object is a singleton providing methods for registration and resolution of different segments.

This class is a singleton and cannot be created directly.

Public Properties

This class has no public properties.
