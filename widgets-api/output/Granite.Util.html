<static> Replaces occurrences of {n} in the specified text with the texts from the snippets.

Example 1 (single snippet):

var text = Granite.Util.patchText("{0} has signed in.", "Jack");
Result 1:
Jack has signed in.

Example 2 (multiple snippets):

var text = "{0} {1} has signed in from {2}.";
    text = Granite.Util.patchText(text, ["Jack", "McFarland", "10.0.0.99"]);
Result 2:
Jack McFarland has signed in from 10.0.0.99.

    The text

  setIFrameMode( [Object topWindow] ) : void
Allows to define if Granite.Util is running in an iFrame and parent window is in another domain (and optionally defin...
Allows to define if Granite.Util is running in an iFrame and parent window is in another domain (and optionally define what would be the top window in that case. This is necessary to use getTopWindow in a iFrame on WebKit based browsers because getTopWindow iterates on parent windows to find the top one which triggers a security exception if one parent window is in a different domain. Exception cannot be caught but is not breaking the JS execution.
Parameters:
  • topWindow : Object
    (optional) The iFrame top window. Must be running on the same host to avoid security exception. Defaults to window.
