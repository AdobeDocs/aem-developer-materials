Package:CQ_Analytics
Class:ActivityStreamMgr
Extends:CQ_Analytics.JSONStore
Clientlib:personalization.stores, personalization.stores.activitystream
Activity stream JSON store. Receives and renders the activities of the currently loaded user.

This class is a singleton and cannot be created directly.
Added in CQ 5.5.
